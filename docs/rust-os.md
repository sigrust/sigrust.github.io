## 推荐课程

### 清华大学OS课程
**rCore-Tutorial**  
[课件](https://learningos.cn/os-lectures/)  
[项目仓库](https://github.com/LearningOS/os-lectures)  
[开放训练营](https://opencamp.cn/os2edu/camp/2026spring)  

清华大学基于Rust的操作系统课程，从零开始构建一个完整的OS内核。课程包括：
- RISC-V架构基础
- 内存管理
- 进程管理
- 文件系统
- 并发与同步

## 推荐教程 

### Writing an OS in Rust
- [Writing an OS in Rust](https://os.phil-opp.com/) - 国际知名的Rust OS教程
  
详细的博客系列教程，一步步教你如何用Rust编写一个小型操作系统。内容包括：
- 启动流程
- VGA文本模式
- CPU异常处理
- 内存管理
- 分页系统
